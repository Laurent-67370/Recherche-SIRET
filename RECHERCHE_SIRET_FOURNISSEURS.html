📚 Guide d'utilisation — SIRET Fournisseurs ABRAPA v5.1

🏠 Présentation générale

Cet outil interroge l'API SIRENE officielle (recherche-entreprises.api.gouv.fr & INSEE) pour enrichir et auditer la base fournisseurs ABRAPA dans Sage FRP 1000. Il fonctionne sans installation ni serveur — un simple fichier HTML à ouvrir dans le navigateur.

Historique des recherches

Les 50 derniers termes recherchés sont affichés en badges horodatés sous le formulaire (à l'instant, 5min, 2h…). Cliquer relance la recherche instantanément. L'historique est conservé entre les sessions grâce au localStorage — il persiste après fermeture du navigateur jusqu'à effacement manuel.

💡 Astuce Sage — Utilisez Tout copier pour coller un tableau champ/valeur dans Excel, puis saisissez la fiche tiers directement sans jongler entre les fenêtres.
📋 Traitement par lot

Lit un fichier CSV, TXT ou Excel, détecte les lignes sans SIRET et interroge SIRENE pour chacune. Les SIRET déjà renseignés sont conservés sans appel API.

Formats acceptés
  • Excel (.xlsx, .xls) — Export direct depuis Sage FRP 1000 ou toute extraction bureautique — Nouveau v5.1
  • CSV / TXT — Séparateur ; ou , auto-détecté, encodage ISO-8859-1 ou UTF-8
  • Colonnes minimales : Nom fournisseur — optionnel : SIRET existant, Code postal

❓ Questions fréquentes
Peut-on renommer le fichier HTML ?

Oui, sans restriction. Le fichier ne contient aucune référence à son propre nom.

L'indicateur API est rouge — que faire ?

Le voyant 🔴 signale que l'une ou les deux APIs SIRENE ne répondent pas. Cliquer sur le voyant puis sur ↻ Tester maintenant pour relancer le diagnostic. Si le problème persiste, vérifier la connexion réseau ou les restrictions du proxy. L'API recherche-entreprises est publique, sans clé API.

L'historique est-il permanent ?

Oui depuis la v5.0. L'historique est stocké dans le localStorage du navigateur : il persiste entre les sessions et après fermeture du navigateur. Les 50 derniers termes sont conservés, chacun horodaté (ex : 5min, 2h, 3j). Cliquer sur Effacer pour réinitialiser.

Comment exporter seulement certains résultats ?

Cliquer sur les cartes résultat souhaitées (ou leurs cases ☑) pour les sélectionner. Un bandeau apparaît en bas avec le bouton 📥 Exporter la sélection. Pour exporter tout, utiliser le bouton 📥 Exporter tout en haut des résultats.

Pourquoi certains fournisseurs sont introuvables ?

Plusieurs raisons possibles : nom abrégé ou différent de la raison sociale officielle, entreprise étrangère (hors registre français), micro-entreprise non inscrite au RCS, ou SIRET effectivement radié. Essayer avec différentes formulations ou compléter avec le code postal.

Le nom trouvé est différent du nom dans Sage — est-ce normal ?

Oui. SIRENE utilise la raison sociale officielle (ex : PHARMAT SAS) alors que Sage peut contenir un nom commercial abrégé (ex : PHARMAT). L'outil ignore les formes juridiques (SAS, SARL…) lors de la comparaison et signale les écarts significatifs avec le statut Nom différent.

La recherche est-elle limitée en volume ?

L'API SIRENE est gratuite et sans clé API. Elle applique un rate-limit : l'outil attend automatiquement 380 ms entre chaque requête (~ 150 fournisseurs/minute). En cas de dépassement (HTTP 429), une pause de 2 s est insérée automatiquement.

La session Batch/Audit est-elle sauvegardée si je ferme l'onglet ?

Oui pour le Traitement par lot et l'Audit : la progression est sauvegardée dans le sessionStorage. Un bandeau de restauration apparaît au rechargement. Fermer complètement le navigateur efface la session.

À quoi sert le bouton 📊 Comparer dans l'Audit ?

Disponible sur les lignes avec données SIRENE (statuts OK, Nom ≠, Fermé, Trouvé/nom). Il ouvre une modale côte à côte avec les valeurs actuelles de Sage et celles de SIRENE pour 10 champs. Chaque différence est signalée par un badge Diff orange. Cliquer Accepter sur chaque champ ou Tout accepter pour valider l'ensemble.

Peut-on utiliser cet outil hors connexion ?

Non. Les 3 modules interrogent l'API SIRENE en temps réel. Une connexion internet est indispensable. En revanche, le fichier HTML s'ouvre directement dans le navigateur sans installation ni serveur.
