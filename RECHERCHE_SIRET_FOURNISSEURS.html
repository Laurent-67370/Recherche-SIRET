📚 Guide d'utilisation — SIRET Fournisseurs ABRAPA

🏠 Présentation générale

Cet outil interroge l'API SIRENE officielle (recherche-entreprises.api.gouv.fr, INSEE / data.gouv.fr) pour enrichir et auditer la base fournisseurs ABRAPA dans Sage FRP 1000. Il peut être renommé librement sans impact sur son fonctionnement.

📋 Traitement par lot

Lit un fichier CSV ou Excel, détecte les lignes sans SIRET (ou avec SIRET nul/zéro) et interroge SIRENE pour chacune. Les SIRET déjà renseignés sont conservés sans appel API.

Formats acceptés

  • CSV / TXT — Séparateur ; ou , auto-détecté, encodage ISO-8859-1 ou UTF-8
  • Colonnes minimales : Nom fournisseur + optionnel SIRET + optionnel Code postal

❓ Questions fréquentes
Peut-on renommer le fichier HTML ?

Oui, sans aucune restriction. Le fichier ne contient aucune référence à son propre nom. Vous pouvez le renommer, le déplacer ou le copier librement.

Pourquoi certains fournisseurs sont introuvables ?

Plusieurs raisons possibles : nom trop abrégé ou différent de la raison sociale officielle, entreprise étrangère (hors registre français), micro-entreprise non inscrite au RCS, ou SIRET effectivement radié. Essayez la recherche unitaire avec différentes formulations du nom.

Le nom trouvé est différent du nom dans Sage — est-ce normal ?

Oui. SIRENE utilise la raison sociale officielle (ex : PHARMAT SAS) alors que Sage peut contenir un nom commercial abrégé (ex : PHARMAT). L'outil signale les écarts significatifs avec le statut Nom différent en ignorant les formes juridiques (SAS, SARL…).

La recherche est-elle limitée en volume ?

L'API SIRENE est gratuite et sans clé API. Elle applique un rate limit : l'outil attend automatiquement 380 ms entre chaque requête (~ 150 fournisseurs/minute). En cas de dépassement (HTTP 429), une pause de 2 s est insérée automatiquement.

La session est-elle sauvegardée si je ferme l'onglet ?

Oui pour les onglets Traitement par lot et Audit : la progression est sauvegardée dans le sessionStorage du navigateur. Un bandeau de restauration apparaît au rechargement. Fermer complètement le navigateur efface la session.

Peut-on utiliser cet outil hors connexion ?

Non. Les 4 modules interrogent l'API SIRENE en temps réel. Une connexion internet est indispensable. En revanche, le fichier HTML s'ouvre directement dans le navigateur sans installation ni serveur.

À quoi sert le bouton « Tout copier » ?

Il copie tous les champs CT_* du résultat sélectionné sous forme d'un tableau champ/valeur (séparateur tabulation). Vous pouvez le coller directement dans Excel ou dans un email pour saisir la fiche 1/3 dans Sage sans revenir à l'outil. Les champs vides sont inclus avec une valeur vide.

Comment exporter uniquement certains résultats ?

Dans l'onglet Recherche unitaire, cliquez sur une ou plusieurs cartes résultat pour les sélectionner (fond orange + case cochée). Un bandeau apparaît en bas avec le compteur de sélection. Cliquez Exporter la sélection pour générer un fichier Excel ne contenant que les lignes choisies. Le bouton Exporter tout en haut exporte toujours l'ensemble des résultats.

À quoi sert la recherche par SIREN ?

Le SIREN (9 chiffres) identifie une entreprise, le SIRET (14 chiffres) identifie un établissement précis. En saisissant un SIREN dans l'onglet Vérifier, l'outil liste tous les établissements de l'entreprise (siège + annexes), utile pour les fournisseurs ayant plusieurs sites.

L'historique des recherches est-il permanent ?

Oui. L'historique est désormais stocké dans le localStorage du navigateur : il persiste entre les sessions, même après fermeture complète du navigateur. Les 50 dernières recherches sont conservées avec leur horodatage. Le bouton Effacer les supprime manuellement.

📊 À quoi sert le voyant statut API dans le header ? v5.1

Ce voyant indique en temps réel si les deux APIs SIRENE sont accessibles. 🟩 Vert = les deux APIs répondent normalement. 🟫 Jaune = une seule est disponible. 🔴 Rouge = aucune n'est joignable (connexion internet à vérifier). Cliquer sur le voyant affiche le détail avec la latence de chaque API et l'heure de la dernière vérification. Le bouton ↻ Tester force un retest immédiat. La vérification automatique s'effectue toutes les 5 minutes.

📊 À quoi sert le bouton « Comparer » dans l'Audit ? v5.1

Ce bouton ouvre une modale de comparaison côte à côte entre les données Sage et celles récupérées de SIRENE. Vous pouvez accepter chaque champ individuellement ou tout accepter en une fois. Les valeurs acceptées sont incluses dans les exports Sage FRP 1000.

Le traitement par lot accepte-t-il les fichiers Excel ? v5.1

Oui depuis la v5.1. Le traitement par lot accepte désormais les mêmes formats que l'Audit : .xlsx, .xls, .csv et .txt. Le format est détecté automatiquement au dépôt du fichier, sans configuration supplémentaire.
