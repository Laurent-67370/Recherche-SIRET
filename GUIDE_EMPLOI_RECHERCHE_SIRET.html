Qu'est-ce que cet outil ?
Une application web autonome permettant de rechercher, vérifier et auditer les numéros SIRET et SIREN des fournisseurs ABRAPA, directement connectée aux bases officielles INSEE SIRENE et data.gouv.fr. Aucune installation requise — s'ouvre directement dans le navigateur.

1

Démarrage rapide

Prérequis

Deux fichiers doivent être placés dans le même dossier sur votre poste :

  • RECHERCHE_SIRET_FOURNISSEURS.html — l'application
  • xlsx.full.min.js — librairie pour les exports Excel

Lancement

Ouvrir le fichier RECHERCHE_SIRET_FOURNISSEURS.html avec votre navigateur (double-clic ou clic droit → Ouvrir avec → Chrome / Edge)
Vérifier que la connexion internet est active — l'outil interroge les bases de données gouvernementales en temps réel
Utiliser les 3 onglets selon votre besoin
💡

Si les exports Excel ne fonctionnent pas, vérifiez que xlsx.full.min.js est bien présent dans le même dossier que le fichier HTML.

2

🔍 Onglet Recherche & Vérification

Un seul champ de saisie intelligent qui s'adapte automatiquement selon ce que vous tapez.

🔤 Mode Nom Recherche par nom

Tapez le nom du fournisseur. Ajoutez un code postal pour affiner. Ex : PHARMAT ou SUPER U

🏢 Mode SIREN 9 chiffres exactement

Retourne la fiche entreprise complète et le bouton pour lister tous les établissements.

🔢 Mode SIRET 14 chiffres exactement

Vérification directe de l'établissement avec toutes ses informations Sage.

Informations retournées

Export depuis la recherche unitaire

Le bouton Export Excel Sage FRP 1000 génère un fichier .xlsx prêt à l'import dans Sage, avec une notice d'import incluse dans un second onglet.

💡

L'historique des recherches est conservé pendant toute la session. Cliquez sur un terme de l'historique pour relancer la recherche.

3

📋 Onglet Traitement par lot

Permet de compléter automatiquement les SIRET manquants à partir d'un fichier exporté de Sage FRP 1000.

Étapes

Importer votre fichier CSV ou Excel extrait de Sage (colonnes minimales : Nom, Code postal, SIRET)
Mapper les colonnes — indiquer quelle colonne correspond au nom, au code postal, au SIRET
Lancer la vérification — l'outil interroge SIRENE pour chaque ligne sans SIRET valide
Exporter les résultats en CSV complet, CSV Sage, ou Excel coloré

Fonctionnalités avancées

  • Pause / Reprise — vous pouvez interrompre et reprendre le traitement sans tout recommencer
  • Sauvegarde de session — la progression est conservée en cas de fermeture accidentelle
  • Barre de progression — avancement en temps réel avec compteur et vitesse

Les lignes avec un SIRET déjà renseigné et valide sont conservées sans modification. Seules les lignes sans SIRET (ou SIRET invalide) sont recherchées.

4

📊 Onglet Audit base fournisseurs

Vérifie en masse la cohérence de toute votre base fournisseurs Sage avec les données officielles SIRENE.

Étapes

Importer l'export complet de votre base fournisseurs Sage (.xlsx)
Mapper les colonnes — CT_Intitule, CT_Siret, CT_CodePostal
Lancer l'audit — vérification automatique de chaque fournisseur
Filtrer et exporter les résultats selon le statut

Les statuts possibles

✅
OK

SIRET confirmé, établissement actif, nom cohérent

🔍
Trouvé par nom

Pas de SIRET dans Sage — trouvé par recherche nom + CP. Action : saisir le SIRET dans Sage

⚠️
Nom différent

SIRET correct mais nom différent entre Sage et SIRENE

🔴
Fermé

L'établissement est administrativement fermé dans SIRENE

❌
Non trouvé

SIRET non confirmé ou introuvable dans SIRENE

⏭️
Sans SIRET

Ni SIRET ni nom suffisant pour rechercher

Mode de vérification dual

  • Avec SIRET → vérification directe : le SIRET de Sage est-il confirmé par SIRENE ?
  • Sans SIRET → recherche automatique par nom + code postal comme fallback

🏢 Liste des établissements d'un SIREN

Accessible depuis la fiche résultat d'une recherche SIREN via le bouton Tous les établissements du SIREN.

✨ Alimenté par l'API INSEE officielle — retourne la liste complète de tous les établissements (ex : 91 établissements pour ABRAPA), avec pagination automatique.

Fonctionnalités de la modale

  • Filtres : Tous / Actifs / Fermés
  • Recherche : par commune, adresse, SIRET, code NAF
  • Siège social : identifié par un badge orange
  • SIRET cliquable : clic pour copier dans le presse-papier
  • Export Excel : liste complète formatée avec statuts colorés
💡

Utile pour identifier quel SIRET correspond à un établissement ABRAPA précis (EHPAD, SSIAD, aide à domicile…) avant de le renseigner dans Sage.

7

💡 Conseils pratiques

Optimiser la recherche par nom

  • Utilisez un nom court plutôt que la raison sociale complète : PHARMAT plutôt que SAS PHARMAT DISTRIBUTION
  • Ajoutez le code postal pour lever les ambiguïtés sur les noms courants
  • En cas de résultat inexact, essayez sans le code postal pour élargir la recherche

Préparer un fichier pour le traitement par lot

  • Exportez depuis Sage : menu Fichier → Exporter → Tiers fournisseurs
  • Colonnes minimales requises : CT_Intitule, CT_CodePostal, CT_Siret
  • Le fichier peut contenir d'autres colonnes — elles seront ignorées

TVA intracommunautaire

Le numéro TVA intra (CT_NumTVAIntracomm) est retourné automatiquement si l'information est disponible dans SIRENE. Les associations non assujetties à la TVA (comme ABRAPA) n'ont pas ce champ renseigné.

En cas de problème de connexion

  • Vérifiez que vous n'êtes pas en mode hors-ligne
  • Si l'export Excel échoue, vérifiez que xlsx.full.min.js est dans le même dossier
  • En cas de message "Erreur HTTP 429", attendez quelques secondes — la limite de requêtes API a été temporairement atteinte

🔑 Clé API INSEE intégrée — valable indéfiniment. En cas d'expiration, contacter le responsable de l'outil pour mise à jour.
